LoveQuest

Github Repository

LoveQuestclose

This project shows off both the front-end and back-end technologies. Uses HTML, CSS, JavaScript, and jQuery on the client-side. LoveQuest is powered by a MySQL database and uses Node.js and Express for its webserver. LoveQuest also uses Sequelize as the ORM and is built using an MVC file structure and SCRUM software development practices. LoveQuest has a lot of functionality such as a map feature which shows a client user location information based on the database, a chat function which allows users to chat with each other in real time, as well as a search functionality and make lovequester functionality which lets users interact with each other's profiles. Overall a very fun project to work on and is currently the pinnacle of my full-stack web developement experience.

Burger Loggerclose

This application allows users to keep track of the burgers they have eaten and allows users to add potential burgers they would like to eat in the future. Burger Logger is also built Node.js and Express on the backend which comprises its webserver. Very fun application to work on and it also made me excited to make more full-stack web applications.
